# Product

A client-side todo list app that runs entirely in the browser. Users can add, complete, edit, and delete tasks. Data is persisted via `localStorage` so tasks survive page refreshes — no backend required.

## Core Features

- Add tasks with title validation (no empty/whitespace-only titles)
- Mark tasks complete/incomplete (toggle)
- Edit task titles inline
- Delete individual tasks or bulk-clear all completed tasks
- Filter tasks by status: All, Active, Completed
- Automatic persistence to `localStorage`

## Language

Requirements and design documents are written in Indonesian (Bahasa Indonesia). Code, comments, and identifiers use English.
